Installation

Requires Angular 17.2+ and Bootstrap 5.3. The schematic does the wiring for you.

Option 1 — ng add recommended

One command installs the dependencies, registers the styles, wires the providers and scaffolds a starter component.

  • Installs bootstrap, bootstrap-icons, @open-rlb/date-tz, @ngx-translate/core and @angular/cdk.
  • Registers the Bootstrap and Bootstrap Icons stylesheets in angular.json.
  • Adds provideRlbBootstrap() to your application providers.
  • Scaffolds a starter component you can render to verify the setup.
  • Copies the bundled Claude skills into .claude/skills/ (date-tz and component guides).

Theming & dark mode

Components follow Bootstrap 5.3 theming. Switch between light and dark by setting the data-bs-theme attribute on <html> (or any subtree) — no extra configuration needed.
