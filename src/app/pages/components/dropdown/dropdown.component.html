Dropdown

Un Dropdown è un componente interattivo che permette di mostrare un elenco di opzioni o azioni quando l'utente clicca su un elemento di attivazione, come un pulsante o un link.

Parameter Type Required Description I/O direction 'up' | 'down' | 'left' | 'right' | 'up-center' | 'down-center' False Controls the opening direction of the dropdown. Default: 'down'.
Note: 'up-center' and 'down-center' are custom extensions not present in native Bootstrap. Input

Dropdown Trigger (Directive)

Applied to a trigger element (button, link, badge). It manages Bootstrap dropdown behavior and exposes additional options.

Parameter Type Required Description I/O offset string False Sets the offset of the dropdown menu relative to the trigger (e.g. "0,10"). Input auto-close 'default' | 'inside' | 'outside' | 'manual' False Controls when the dropdown should close:
- default: closes when clicking inside or outside.
- inside: closes only on inside clicks.
- outside: closes only on outside clicks.
- manual: requires programmatic closing. Input status-changed 'show' | 'shown' | 'hide' | 'hidden' False Event emitted when dropdown visibility changes (mapped from Bootstrap events). Output

Dropdown Container

The container that wraps dropdown items. Can be a <ul rlb-dropdown-menu> for list-based menus, or <rlb-dropdown-container> for custom content.

Parameter Type Required Description I/O placement 'left' | 'right' False Menu alignment relative to trigger. Input placement-sm 'left' | 'right' False Responsive placement for small screens. Input placement-md 'left' | 'right' False Responsive placement for medium screens. Input placement-lg 'left' | 'right' False Responsive placement for large screens. Input placement-xl 'left' | 'right' False Responsive placement for extra large screens. Input placement-xxl 'left' | 'right' False Responsive placement for extra extra large screens. Input

Dropdown Item

Individual items inside a dropdown list (<li rlb-dropdown-item>). Supports different modes: normal item, header, divider.

Parameter Type Required Description I/O active boolean False Highlights the item as active. Input disabled boolean False Disables the item. Input header boolean False Renders the item as a dropdown section header. Input divider boolean False Renders a visual divider between items. Input link string False Route link reference. If populated renders element as html anchor and assigns routerLink Input text-wrap boolean False Applies bootstrap text-wrap and text-break classes . Input
