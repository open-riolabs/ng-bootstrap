Dropdown

Un Dropdown è un componente interattivo che permette di mostrare un elenco di opzioni o azioni quando l'utente clicca su un elemento di attivazione, come un pulsante o un link.

Parameters Type Required Description I/O direction 'up' | 'down' | 'left' | 'right' | 'up-center' | 'down-center' False Aggiungi per impostare la direzione verso quale si aprirà il menù dropdown. Default: 'down' Input
