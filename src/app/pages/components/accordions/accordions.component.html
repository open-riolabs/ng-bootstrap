Accordion

Accordion è un componente che consente di organizzare il contenuto in sezioni a fisarmonica, permettendo all'utente di espandere e contrarre ogni sezione.

Parameters Type Required Description I/O flush boolean False Add to remove some borders and rounded corners to render accordions edge-to-edge with their parent container. Default: false Input alwaysOpen boolean False Add to make accordion elements stay open when another element is opened. Input id string False Add to make the accordion elements have an id Input

Accordion Item

L'Accordion item è un singolo elemento del componente Accordion. Aggiungi più Accordion Item dentro il tuo Accordion per creare più sezioni

Parameters Type Required Description I/O name string False Add to make the accordion elements have an name Input expanded boolean False Add to ensure that a section of the accordion always remains open Input class string False Add to cause a CSS class to be applied to the accordion Input style string False Add to have a style applied to the accordion Input status VisibilityEvent False Add to apply change state VisibilityEvent Input statusChange EventEmitter<VisibilityEvent> False Take to receive a VisibilityEvent status change Output
Accordion Header

Aggiungi un Accordion Header all'interno del Accordion Item per definire la parte che verrà cliccata dall'utente

Parameters Type Required Description I/O Nessun Parametro
Accordion Body

Aggiungi un Accordion Body all'interno del Accordion Item per definire la parte interna del componente che sarà visibile quando l'Accordion sarà aperto
