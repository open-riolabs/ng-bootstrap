Card

Una Card è un contenitore flessibile e versatile usato per raggruppare contenuti e informazioni, come testi, immagini, link e pulsanti, in un formato visivamente strutturato

Parameters Type Required Description I/O align TextAlignment False Add to apply text alignment. Default: 'left' Input overlay boolean False Add to Transform an image into the background of a card and overlay the text of your card. Input background Color False Add to apply a color to the background. Input border Color False Add to apply a color to the border. Input

Card Image

Inserisci una Card Image all'interno della tua Card per definire il corpo principale del componente che avrà un immagine principale come background e testo sovrapposto.

Parameters Type Required Description I/O position 'top' | 'bottom' False Add to define the position of the image. Default: 'top' Input
