Scrollspy

Lo Scrollspy è un componente che traccia la posizione di scorrimento dell'utente nella pagina e aggiorna automaticamente i link di navigazione per evidenziare la sezione corrente.

Parameter Type Required Description I/O rlb-scrollspy string True CSS selector pointing to the navigation container that contains anchor links (e.g. #nav). Input height string True Scroll container height. Accepts any valid CSS value (e.g. 150px, 100%). Input scroll-smooth boolean False Enables smooth scrolling animation when clicking on anchor links. Input scroll-root-margin string False Custom root margin used to determine the visible area for activating sections (e.g. 0px 0px -50%). Input scroll-threshold number[] False Array of intersection thresholds (from 0 to 1) used to fine-tune when a section becomes active. Input scroll-change EventEmitter<void> False Fires whenever the active section changes. Useful for triggering custom behavior (e.g. UI updates, logging, etc.). Output
