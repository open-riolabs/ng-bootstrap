Button

Un Button è un elemento interattivo che consente agli utenti di eseguire azioni, come inviare moduli, avviare processi o navigare tra sezioni

FAB (Floating Action Button)

Parameters Type Required Description I/O color Color False Defines the color of the floating action button. Input size 'xs' | 'sm' | 'md' | 'lg' False Defines the size of the floating action button. Input disabled boolean False Disables the floating action button when set to true. Input outline boolean False Applies outline style instead of a filled button background. Input position 'br' | 'bl' | 'tr' | 'tl' | undefined False Sets the screen position of the floating action button.
br – bottom right (default)
bl – bottom left
tr – top right
tl – top left
undefined – FAB behaves like a normal inline button (no fixed position) Input
