Calendar

A powerful and flexible calendar component for displaying events in Month, Week, or Day views.

Parameters Type Required Description I/O [view] 'month' | 'week' | 'day' No The current view of the calendar. Default is 'week'. Input [events] CalendarEvent[] No List of events to display. Input [current-date] IDateTz No The current date focused in the calendar. Input [loading] boolean No Whether to show a loading state. Input [show-toolbar] boolean No Whether to show the top navigation toolbar. Default is true. Input [layout] CalendarLayout No Configuration for row heights and header heights. Input (date-change) CalendarChangeEvent No Emitted when the current date changes. Output (view-change) CalendarChangeEvent No Emitted when the view changes. Output (event-click) CalendarEvent No Emitted when an event is clicked. Output
