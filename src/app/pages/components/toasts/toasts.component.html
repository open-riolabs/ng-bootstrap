Toast

Un componente toast è un messaggio breve e non invasivo che appare temporaneamente per fornire un feedback rapido all'utente, senza interrompere l'attività in corso.

Parameters Type Required Description I/O containerId string True Specifies a unique toast's container identifier, which should be defined in rlb-toast-container component Input name string True Specifies a unique toast identifier, which should be defined in the app module's providers. Input data ToastData True Basic configuration required for the toast. Input options ToastOptions False Optional advanced configuration for the toast. Input
