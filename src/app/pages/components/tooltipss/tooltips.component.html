Tooltip

Il Tooltip è un piccolo riquadro informativo che appare quando l'utente passa il cursore o si sofferma su un elemento, fornendo dettagli aggiuntivi o suggerimenti contestuali.

Parameters Type Required Description I/O tooltip string | null | undefined True Input tooltip-placement 'top' | 'bottom' | 'left' | 'right' False Input tooltip-class string False Input tootltip-html boolean False Input

Popover

Il Popover è un elemento di interfaccia simile al Tooltip, ma più ampio e configurabile, che appare sopra un elemento quando l'utente ci clicca o passa il mouse.

Parameters Type Required Description I/O popover string | undefined True Input popover-placement 'top' | 'bottom' | 'left' | 'right' False Input popover-class string False Input popover-title string False Input
