Alerts

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Examples #

Alerts are available for any length of text, as well as an optional close button. For proper styling, use one of the eight required contextual props (e.g., color="success"). For inline dismissal, use the prop dismissible

Dismissing #

You can make alerts dismissible (i.e., have a close button) by adding the prop dismissible. This adds extra padding to the right of the alert and positions the close button in the top right corner. Be sure to add the .fade and .show classes to animate the alert when it is closed.
