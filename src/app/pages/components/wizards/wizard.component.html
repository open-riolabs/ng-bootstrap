There is no dedicated wizard component — a wizard is just a rlb-carousel used as a controlled stepper. Each rlb-carousel-slide is one step; you bind [current-slide] to a signal and update it from the footer buttons, while (slide-count) tells you how many steps there are. Validate one step at a time to gate the Next button. See the full Carousel API for all options.
