Color

Color è un componente che consente agli utenti di selezionare un colore e visulizzarlo su barra

Parameters Type Required Description I/O disabled boolean False Disables the color picker input when set to true. Input readonly boolean False Makes the color picker read-only. User can’t change the value. Input size 'small' | 'large' False Controls the size of the input. Can be 'small' or 'large'. Input userDefinedId string False Custom ID for the input. If not set, an ID is auto-generated. Input
