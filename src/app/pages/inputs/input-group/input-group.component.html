Input-group

Input-group è un wrapper attorno a uno o più componenti input che fornisce raggruppamento di campi e supporto per validazione esterna tramite rlb-input-validation

Parameter Type Required Description I/O text boolean False Enables input-group-text mode for displaying plain text instead of input fields. Input validate boolean False Enables external validation rendering. Collects errors from child rlb-input components and passes them to an optional rlb-input-validation inside the group. Input size 'small' | 'large' False Controls group size by applying Bootstrap’s input-group-sm or input-group-lg classes. Input
