Input

Si tratta di un input Angular universale ed estensibile, racchiuso in un componente form-friendly. Supporta tutti i tipi di input HTML (testo, email, numero, password, datetime-local, ecc.)

Parameters Type Required Description I/O type string False Input type: text, email, number, password, search, tel, url, etc. Input disabled boolean False Disables the input field Input readonly boolean False Makes the input read-only Input placeholder string False Placeholder text Input name string False HTML input name attribute Input size string False 'small' or 'large' to control input size Input max / min / step number False Constraints for numeric inputs Input dateType string False For datetime-local input:
'string', 'number', 'date', or 'date-tz' to define the output format. Input timezone string False Timezone used for parsing date-tz inputs (default: 'UTC') Input id string False User-defined ID for the input element Input
