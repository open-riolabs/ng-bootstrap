Datalist

Datalist è un componente che consente all'utente di immettere testo con la possibilità di suggerimenti automatici, ma non limita la selezione solo all'elenco proposto.

Parameters Type Required Description I/O disabled boolean False Disables the input when set to true. Input readonly boolean False Makes the input read-only. Prevents user editing. Input placeholder string False Sets the placeholder text inside the input. Input size 'small' | 'large' False Changes the input size using Bootstrap styles. Input id string False Custom ID for the input. If not set, it is auto-generated. Input
