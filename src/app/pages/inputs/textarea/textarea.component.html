Textarea

Textarea è un componente UI che permette di inserire testo a più righe in un campo, spesso utilizzato in moduli web per input di testo più lunghi, come commenti, descrizioni o messaggi.

Parameter Type Required Description I/O disabled boolean false Disables the textarea, making it non-interactive. Input readonly boolean false Makes the textarea read-only, preventing user edits while keeping it focusable. Input placeholder string false Text shown when the textarea is empty. Input size 'small' | 'large' false Changes the visual size of the textarea using Bootstrap's sizing classes. Input rows number false Specifies the number of visible text lines. Default is 3. Input id string false Custom identifier for the textarea element. If not provided, an auto-generated ID is used. Input
