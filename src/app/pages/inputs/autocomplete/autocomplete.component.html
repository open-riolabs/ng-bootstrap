Autocomplete

Autocomplete è un componente che combina un input di testo con una casella di riepilogo, consentendo agli utenti di filtrare un elenco di opzioni in base agli elementi che corrispondono a una query.

Parameters Type Required Description I/O disabled boolean False Specifies whether the component is disabled (default is false). Input readonly boolean False Specifies whether the component is read-only (default is false). Input loading boolean False Indicates whether a loading spinner is displayed. Used for asynchronous operations (default is false). Input max-height number False Sets the maximum height of the dropdown list (default is 200px). Input placeholder string False Text displayed in the input field when it is empty. Input autocomplete AutocompleteFn False A function that returns an array of autocomplete options, taking a query string as an argument (can return a Promise or Observable). Input type string False Specifies the input field type: `text`, `email`, `number`, `password`, `search`, `tel`, `url` (default is `text`). Input size small | large False Specifies the size of the input field: `small`, `large`, or `undefined` (default). Input chars-to-search number False Minimum number of characters required in the query before search starts (default is 3). Input menu-max-width number False Maximum width of the autocomplete dropdown menu (default is 400px). Input id string False User-defined ID for the component, automatically generated if not provided. Input open-on-focus boolean False When true, opens the dropdown on input focus even if the field is empty (default is false). Input initial-suggestions AutocompleteItem[] False Pre-loaded suggestions shown on focus when the field is empty. Once the user starts typing, the autocomplete function takes over (default is []). Input selected EventEmitter False Event emitted when an item is selected from the autocomplete list. Output
AutocompleteItem interface
Property Type Required Description text string True Label displayed in the dropdown and filled into the input on selection. value string True Unique identifier stored as the form control value. iconClass string False CSS class for an icon rendered before the label (e.g. flag icons). disabled boolean False When true, the item is visible in the dropdown but cannot be selected (rendered with Bootstrap's .disabled style and cursor: not-allowed). data T False Arbitrary extra payload attached to the item, accessible from the selected output.
