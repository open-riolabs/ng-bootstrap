Checkbox

Checkbox è un componente che consente agli utenti di selezionare uno o più elementi da un set. Le checkbox possono attivare o disattivare un'opzione.

Parameters Type Required Description I/O disabled boolean False Disables the checkbox input when set to true. Input readonly boolean False Makes the checkbox read-only. It will be visible but not interactable. Input indeterminate boolean False Enables the indeterminate visual state when value is undefined or null. Input id string False User-defined ID for the checkbox element. If not provided, it will be auto-generated. Input
