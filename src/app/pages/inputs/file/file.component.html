File

Permette di caricare un file

Parameters Type Required Description I/O disabled boolean False Disables the file input when set to true. Input readonly boolean False Makes the input read-only (not functional for file inputs in most browsers). Input multiple boolean False Allows selecting multiple files if set to true. Input accept string False Specifies accepted file types (e.g., .jpg,.png or image/*). Input size 'small' | 'large' False Adjusts the input size. Input id string False Custom ID for the input. If not provided, one will be auto-generated. Input
