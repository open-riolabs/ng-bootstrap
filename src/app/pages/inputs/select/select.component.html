Select

Select è un component che permette all'utente di scegliere un'opzione da un elenco. Funziona come un menu a tendina o un dropdown, offrendo una lista di opzioni tra cui l'utente può selezionare.

Parameter Type Required Description I/O placeholder string False Placeholder text shown when no option is selected (used only in single-select mode). Input size "small" | "large" False Visual size of the select component (CSS class-based). Input disabled boolean False Disables the select input. Input readonly boolean False Prevents changes to selection but still allows scrolling and interaction (note: HTML native readonly is not fully supported on <select> elements). Input multiple boolean False Allows multiple selection if true. Input display number False Sets the size attribute of the <select> element (number of visible options). Input id string False Custom identifier for the select element. Auto-generated if not provided. Input
