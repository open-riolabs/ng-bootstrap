Switch

Un componente switch (o toggle) è un elemento del UI che permette all'utente di scegliere tra due stati: acceso o spento, abilitato o disabilitato, ecc. È essenzialmente un interruttore visivo che consente di attivare o disattivare un'opzione o una funzionalità.

Parameter Type Required Description I/O disabled boolean false Disables the switch toggle, preventing user interaction. Input readonly boolean false Makes the switch read-only. Visually enabled but not changeable. Input size 'small' | 'large' false Changes the visual size of the switch toggle. Input id string false Custom identifier for the switch input. Auto-generated if not provided. Input
